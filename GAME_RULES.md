# 🎮 Game Rules

## How to Play

Predict PSL match winners and earn points.

## Prediction Rules

### Timing
- **Opens:** Exactly 50 minutes before match start
- **Closes:** At match start time
- **Window:** 50-minute prediction period
- **After closing:** Cannot change prediction

### Edits
- Maximum 2 edits per match
- Only final prediction counts

## Scoring

### Points
- ✅ Correct prediction: **+2 points**
- ❌ Wrong prediction: **0 points**
- 🚫 No prediction: **0 points**
- ⏸️ Abandoned match: **+1 point** (if you predicted)

### Streak Bonus
- Get 3 correct in a row: **+1 bonus point**
- Wrong prediction: Streak resets to 0
- No prediction: Streak resets to 0
- Abandoned match: Streak unchanged

## Leaderboard

- Ranked by total points (highest first)
- Same points = same rank (no tie-breaking)
- Updates automatically when results are added

## Examples

### Perfect Streak
- Match 1: Correct → 2 points (streak = 1)
- Match 2: Correct → 2 points (streak = 2)
- Match 3: Correct → 2 + 1 bonus = 3 points (streak = 3)
- **Total: 7 points**

### Broken Streak
- Match 1: Correct → 2 points (streak = 1)
- Match 2: Correct → 2 points (streak = 2)
- Match 3: Wrong → 0 points (streak = 0)
- **Total: 4 points**

### Abandoned Match
- Match 1: Correct → 2 points (streak = 1)
- Match 2: Correct → 2 points (streak = 2)
- Match 3: Abandoned → 1 point (streak = 2, unchanged)
- Match 4: Correct → 2 + 1 bonus = 3 points (streak = 3)
- **Total: 8 points**

## Tips

- Predict early to avoid missing the window
- Use your 2 edits wisely
- Never miss a prediction - even 1 point helps
- Maintain streaks for bonus points

Good luck! 🏏🔥
